# c-sharp-web-application
Web portal with news area and mini-shop

Technologies used in this project:

 - Microsoft .NET Framework;
 - ADO.NET;
 - Entity Framework;
 - MembershipProvider;
 - AjaxControlToolKit;
 - WebService;
 - Web server (IIS);
 - System.Net;
 - System.Net.Mail;
 - ReportViewer with RDLC.
